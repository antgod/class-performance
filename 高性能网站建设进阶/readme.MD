##js性能

1. 理解ajax性能:
    - 大部分时间浪费在循环上
    - ajax使app更快,但是开发周期更长
    - 运行程序最大的开销是DOM而非JS

2. 网络性能

    - 可伸缩的comet

    - 超越gzip压缩

    - 图像优化

    - 划分主域

    - 尽早刷新文档的输出

3. 浏览器性能

    - 减少使用iframe

    - 简化css选择符

        - 最右边优先

        - 避免使用通配\元素

        - 尽可能使用.class #id

        - 尽可能依靠继承

        - 回流时间-修改样式表而不是元素样式
